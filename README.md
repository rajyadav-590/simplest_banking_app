# 🏦 ABC Bank System (Python CLI Project)


\

---

## 📌 Overview

ABC Bank System is a simple **Command Line Banking Application** built using Python.
It allows users to perform basic banking operations like checking balance, depositing money, and withdrawing money.

This project is ideal for beginners who want to strengthen their understanding of **core Python concepts**.

---

## 🚀 Features

* 🔍 Check account balance
* 💰 Deposit money
* 💸 Withdraw money
* ⚠️ Input validation (handles invalid/negative input)
* 🔁 Continue or exit option
* 🛑 Clean and user-friendly interface

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **Interface:** Command Line (CLI)

## 💡 How It Works

1. User selects an option from the menu
2. Inputs amount for deposit/withdraw
3. Program validates input
4. Balance updates accordingly
5. User chooses to continue or exit

---

## ⚠️ Limitations

* ❌ No data persistence (resets after exit)
* ❌ Single user only
* ❌ No authentication system
